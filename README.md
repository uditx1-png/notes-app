# 📝 Notes App (Full Stack)

A full-stack Notes Application built using React, Node.js, Express, and MongoDB.

## 🚀 Features
- 🔐 User Authentication (JWT)
- ➕ Add Notes
- 📄 View Notes
- ❌ Delete Notes
- 🌐 Deployed on Render

## 🛠️ Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Deployment: Render
